About Me

Professional Summary Highly skilled data scientist with extensive experience in applying data science techniques to health, physical, chemical, and biological datasets. Proficient in Python and R programming, with expertise in AI and machine learning (classification, clustering, regression, model selection, preprocessing, dimensionality reduction, and neural networks, leveraging large language models (LLMs using LangChain, Hugging Face)). Adept at QSAR modeling, Read-Across, PBPK, and IVIVE, with a proven ability to map and merge diverse datasets to address complex problems. Additionally, possess strong teaching experience in health informatics at the university level. Excellent communicator with robust problem-solving abilities, teamwork, adaptability, and time management skills. I consider myself a quick learner, adept at adapting to new technologies with ease.

Projects

    Developed QSAR models for cytotoxicity prediction using XGBoost, LightGBM, Random Forest, and deep learning on Tox21 and ToxCast datasets.
  • Project 2: Chemical Carcinogenicity & Mutagenicity Prediction
    Used Graph Neural Networks and deep learning to analyze molecular graph data for toxicological endpoints.
  • Project 3: Chemical Similarity Calculator Tool
    Summary: User-friendly tool for calculating chemical similarity between molecules.
    Tasks:
    • Load a base dataset of chemicals (CSV file with SMILES, DTXSID, NAME, etc.).
    • Accept user input for chemical identifiers (DTXSID, NAME, or SMILES).
    • Allow users to select fingerprint types (ECFP, MACCS, Daylight, PubChem) and similarity metrics (Tanimoto, Cosine, Euclidean, CityBlock, Hamming).
    • Calculate similarity scores between user-input chemicals and the base dataset.
    • Display progress and allow saving results.
    Output:
    • A CSV file listing each input chemical and its most similar chemicals from the base dataset, along with similarity scores and the method used.
    • The output file can be customized and saved by the user.
